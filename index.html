Usage Notes

The page now does the filtering for you, but these checks still matter before you wire an API into the main site.

Practical checks

Watch licensing

Some APIs are free only for non-commercial, educational, or development use. The cards flag that where it matters.

Cache aggressively

Public free APIs often rate limit. A simple local cache or backend proxy usually saves you fast.

Hide keys properly

If an API needs a key, keep it off the frontend when possible and route requests through your backend.
